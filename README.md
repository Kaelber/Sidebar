## Sidebar+

WebExtension that imprement Sidebar into browser.

### Install from source

**Chromium based browsers:**

- Open chrome://extensions/
- Check "Developer mode"
- Click "Install unpacked extension" button

**Firefox:**

- Open about:debugging
- Click "Load Temporary Add-on"
